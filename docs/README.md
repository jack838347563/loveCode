## 项目简介

java 开发参考手册, awesome java. , 目前更多的是当成个人总结备忘工具来使用.

好记性不如烂笔头,几年开发下来,愈加发现记忆力不行.还好有这些笔记类工具可以代替大脑记忆,减轻负担.
